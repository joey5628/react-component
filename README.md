## 一个基于React的移动端组件集

### Overview
使用React, React-Router-v4, Redux搭建H5移动端项目架构，实现Code spliting 及SwipeAction, Modal, Picker, SearchBar等常用UI组件。

### Features
- React
- React-router v4
- Redux
- Babel
- Webpack

### Getting Started
```shell
$ git clone git@github.com:zhangyi5628/react-component.git
$ cd react-component
$ npm install
```

### Run
```shell
$ npm start
```

### Example
http://localhost:4000/#/demo
